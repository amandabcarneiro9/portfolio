DAS Travel App

Foi um projeto final de um Bootcamp. O objetivo era oferecer uma aplicação de página única que cobrisse algumas tecnologias que foram ensinadas durante o Bootcamp, como PHP, Laravel, React e Javascript. A aplicação oferece aos usuários uma breve experiência durante a viagem para uma dessas 3 cidades (Varsóvia, Rio de Janeiro e Santorini) através do aprendizado de algumas conversas simples no idioma local ou tendo conselhos sobre lugares para visitar. Através deste projeto, estávamos à frente de tudo, e como equipe, criamos um banco de dados que foi consumido pela aplicação, também usamos API para mapas e gravatar para usar como perfil. Nesta aplicação, o usuário pode criar uma conta e salvar suas frases favoritas e lugares para visitar, editar o perfil, enviar e-mails para nós e recuperar sua senha se necessário. Para a organização, usamos técnicas ágeis e Trello como scrum.
